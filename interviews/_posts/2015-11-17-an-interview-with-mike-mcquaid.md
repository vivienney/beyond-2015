Sam Billingham was joined by Mike McQuaid, Senior Software Engineer at Github and host of the Github Patchwork workshop at Beyond. They discuss the version control system Git and how it integrates with the Github Platform. Mike shares some of his background on landing a job at Github and gives some advice to people just getting started with code.
